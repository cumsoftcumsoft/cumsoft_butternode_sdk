Welcome to HTML!

HTML is also known as Hyper Text Markup Language. It is a used to design web pages that can be displayed on the browsers. Hypertext generates the link between the web pages and markup language.

Basic Structure of HTML Page:

<!DOCTYPE html>
<HTML>
 <HEAD>
  <TITLE>Title Here</TITLE>
 </HEAD>
 <BODY>
  <H2>HTML Structure</H2>
  <P>Tell something about the Web page here.</P>
 </BODY>
</HTML>
